Features

BQ Tree Browser

PDF Plan Take-off

Open PDF drawings directly, set your scale, and measure areas, perimeters, and linear dimensions with polygon and polyline tools.

Cloud Sync

Your project database and measurements sync securely across devices. Pick up where you left off on any machine.
